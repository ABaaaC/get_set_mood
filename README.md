# Recuirements

lite-server
hardhat

Cheatsheet of main commands:

```
npm install
npm start # in frontend dir
npx hardhat node
lite-server
npx hardhat run scripts/deployMood.js --network goerli
npx hardhat run scripts/deployMood.js --network localhost

npx hardhat compile
npx hardhat clean
```
